# FiterAI

FiterAI is a web-first virtual try-on app that uses a live camera feed and clothing inputs to preview how garments could look on a person in real time.

Current prototype features:
- webcam preview
- garment upload
- live fit overlay
- fit adjustment controls
- snapshot capture and saved looks

Long term, the project aims to support more realistic fitting, shopping-page garment detection, and stronger AI-based rendering.
